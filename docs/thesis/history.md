# 历年论文速览

2022年5月份之前，每批考试会有2个论文主题可选，也就是`2选1`，在2022年5月份之后，每批论文考试只有一个题目。其中2023年的11月有四批考试

## 历年题目

| 2024年 | `进度管理`、`成本管理` |   -   |
| 2023年下 | `合同管理`、`资源管理`、`项目工作绩效域`、`干系人管理` |  - |
